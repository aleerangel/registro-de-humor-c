# Registro de Humor

## 📝 Sobre
Trabalho da disciplina de **Estrutura de Dados I** no curso de Ciência da Computação - UFES.  
Aplicação em linguagem C que registra o humor diário de um usuário com base em uma estrutura de lista duplamente encadeada.  
Permite armazenar, buscar, remover e analisar os registros com base em critérios como frequência do humor e média de notas do dia.

## 🎮 Como rodar

Para compilar e executar:

```bash
# No terminal, dentro da pasta do projeto:
make
````

💡 Certifique-se de ter o `gcc` instalado e estar usando um ambiente compatível com **Makefile** (como Git Bash ou WSL no Windows).

## 🚀 Tecnologias

![C](https://img.shields.io/badge/C-00599C?style=for-the-badge\&logo=c\&logoColor=white)

## 📁 Organização do Código

* `main.c`: Interface com o usuário e menu principal.
* `ListaDuplamenteEnc.[c|h]`: Implementação da lista duplamente encadeada.
* `RegistroDeHumor.[c|h]`: Definições de tipo, leitura e impressão dos registros.

## 🧠 Conteúdos Trabalhados

* Manipulação de listas duplamente encadeadas
* Uso de TADs em C
* Enumerações e strings
* Leitura estruturada de dados do usuário
* Separação modular de arquivos `.c` e `.h`
* Uso de Makefile para automação de build

## 🧼 Limpar arquivos de build

```bash
make clean
```
